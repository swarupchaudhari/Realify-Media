# Realify-Media

Realify Media is a next-generation social platform designed for students, professionals, freelancers, creators, and entrepreneurs.

## Vision

To create a trusted community where users can learn, connect, collaborate, and grow together.

## Features (MVP)

* Authentication
* User Profiles
* Posts
* Follow System
* Real Reels
* Communities
* Messaging

## Tech Stack

Frontend:

* React.js
* Tailwind CSS
* Vite

Backend:

* Node.js
* Express.js

Database:

* MongoDB
